# msp-infrastructure Role

## Description
MSP platform role for msp-infrastructure functionality.

## Requirements
- Ansible 2.10+
- Target systems: RHEL/CentOS 7-9, Ubuntu 18.04-22.04

## Role Variables
See `defaults/main.yml` for available variables.

## Dependencies
None.

## Example Playbook
```yaml
- hosts: servers
  roles:
    - msp-infrastructure
```
